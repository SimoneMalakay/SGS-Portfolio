# Simone Greco Solutions - Portfolio Website

A minimal, professional portfolio website for Simone Greco Solutions (SGS), showcasing 360 IT consultancy services from field to cloud.

## Features

- **Minimal UX Design**: Clean, focused interface that clearly communicates the business mission
- **Responsive Layout**: Works seamlessly across all devices
- **Smooth Navigation**: Single-page design with smooth scrolling
- **Professional Content**: Highlights real estate management solutions, Chronostack leadership, and general IT consultancy

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript

## Deployment

This website is configured for easy deployment to Netlify.

### Deploy to Netlify

1. Connect your GitHub repository to Netlify
2. Netlify will automatically detect the site settings
3. The site will deploy automatically on every push to the main branch

### Manual Deployment

1. Build the site (no build step required for static HTML)
2. Deploy the `index.html` and associated files to Netlify

## Local Development

Simply open `index.html` in a web browser or use a local development server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```
